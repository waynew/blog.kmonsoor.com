kmonsoor's online hubhttp://blog.kmonsoor.com/my internet cornerSun, 29 Jan 2012 12:48:00 +0600Windows 7 Enterprise/Ultimate “K” to “EN” (aka English): Conversion Step-by-stephttp://blog.kmonsoor.com/convert-windows-7-k-to-en/<p>Ended up with Windows 7 Enterprise in Korean language? What you are seeing, all are in Korean or any other foreign language? I am here to share my own experience. It may help you&nbsp;out.</p> <p>As I was working for a South Korean conglomerate, I had access to Windows 7 Enterprise K. So, I have to face a lot of trouble to make the conversion to English. As, there is no Microsoft&#8217;s official help available, I have to find it myself. It would be great if I can share this with proper screen-shots of Korean Windows. But as I had already converted, I could not. Sorry about&nbsp;that.</p> <p>First of all, it will be best if you can install Windows 7 English version directly.<br /> If not possible at all, like your company have license only to use Korean version, or you purchased your laptop or <span class="caps">PC</span> in Korea. In those cases this post may&nbsp;help. </p> <p><strong>I am assuming&nbsp;that: </strong></p> <ul> <li>You have a valid license key or <span class="caps">OEM</span> <span class="caps">VLK</span>(Volume License Key) of Windows 7, either for Enterprise or for Ultimate&nbsp;edition. </li> <li>If you don&#8217;t have Enterprise or Ultimate edition, but you have Home or Professional addition, the following method probably won&#8217;t&nbsp;work. </li> </ul> <h2>Steps:</h2> <ul> <li>Install <strong>Windows 7 K</strong> on your <span class="caps">PC</span>; <span class="caps">DON</span>&#8217;T change any settings if you don&#8217;t understand what it is&nbsp;saying.</li> <li>Don&#8217;t install any anti-virus yet. Or any other system&nbsp;software.</li> <li>Validate and register&nbsp;your license.</li> <li>Go to Windows update.<br /> And update your Windows to Service pack 1. Please <strong>avoid</strong> updating Internet Explorer 8 to 9. Uncheck&nbsp;it.</li> <li>Now Windows will restart&nbsp;automatically.</li> <li>Again, Go to Windows update. In the <strong>optional</strong> section, you&#8217;ll will see a long list of language pack. Select <em>English</em>. </li> <li>Complete update. Again&nbsp;restart.</li> <li>If your <em>Windows 7 K</em> is not converted to <em>Windows 7(English)</em> then from <strong>Language settings</strong>, choose <em>English</em> as display&nbsp;language.</li> <li>Again, restart. Now it should be in English. Some text still may be in Korean. Don&#8217;t care for it. Your life will be quite okay with&nbsp;that.</li> <li>Now you install whatever software you&nbsp;want.</li> </ul> <h2>Postscript</h2> <ul> <li>if now you are in trouble with <span class="caps">MS</span> Office&#8217;s Korean package, check out <a href="install-english-language-on-microsoft-office-2007-2010/index.html">my relevant&nbsp;post.</a> </li> <li>After completion, you may find trouble using Microsoft&#8217;s <strong>Internet Explorer 9</strong>. I don&#8217;t know why <strong>Internet Explorer 9</strong> don&#8217;t work properly on Windows 7.<br /> If you face problem after installing it like &#8220;<span class="caps">I.E.</span> is not responding&#8230;.&#8221;, &#8220;<span class="caps">I.E.</span> closed unexpectedly&#8221; etc. <strong>uninstall that</strong>.  </li> </ul> <p>But how? <a href="http://blog.kmonsoor.com/uninstall-ie-9">Please visit my relevant&nbsp;post.</a></p>Khaled MonsoorSun, 29 Jan 2012 12:48:00 +0600tag:blog.kmonsoor.com,2012-01-29:convert-windows-7-k-to-en/englishkoreanunofficialwindowsconfigurationKoreanSouth KoreatutorialWindows 7conversion
